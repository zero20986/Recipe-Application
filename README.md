# Recipe-Application

Application Name: Recipe Manager 

Application Description: This application allows you to create, view, and manage recipes. You can add new recipes, view a list of all your 
recipes, filter recipes by ingredient name, food group, and maximum calories, and open a recipe's details window. 

How to Use: 
To create a new recipe, click the "Add Recipe" button. 
Enter the recipe's name, ingredients, and instructions. 
Click the "Save" button to save the recipe. 
To view a list of all your recipes, click the "View Recipes" button. 
To filter recipes, enter the desired criteria in the "Ingredient Name", "Food Group", and "Maximum Calories" textboxes. 
Click the "Filter" button to filter the recipes. 
To open a recipe's details window, select the recipe from the list and click the "Open Recipe" button. 

Dependencies: 
This application requires the following dependencies: 
Windows Presentation Foundation (WPF) 
Microsoft .NET Framework 4.8 

To Run: 
Download the application's source code. 
Open the solution file in Visual Studio. 
Build and run the application.
